vArmyKnife: Variant Army Knife

v2.1.82 Revised Tue Aug 14 12:33:15 EDT 2018

The vArmyKnife software package is a fast, efficient, and portable multifunction toolkit designed to assist in the analysis, quality control, and data management of whole-genome and whole-exome variant datasets.

The entire vArmyKnife toolkit can be used in almost any operating system that supports java. For more information visit the vArmyKnife project page.

Download and Installation:

You can download the most recent version of vArmyKnife here (https://github.com/hartleys/QoRTs/releases). Simply download the vArmyKnife.zip or .tar.gz file and extract it to your preferred location.

tar xvzf varmyknife.tar.gz /my/install/directory/

If you are running either Linux or OSX, you can install the software onto your PATH using the command:

export PATH=/my/install/directory/bin/:$PATH
